# PythonCourse10Project

Консольное приложение на Python для обработки банковских транзакций: загрузка данных, фильтрация, сортировка и анализ операций.

---

##  Возможности

Программа позволяет:

- загружать транзакции из:
  - JSON
  - CSV
  - XLSX
- фильтровать операции:
  - по статусу (`EXECUTED`, `CANCELED`, `PENDING`)
  - по валюте (например, только RUB)
  - по ключевому слову в описании
- сортировать операции по дате
- безопасно отображать данные (маскирование карт и счетов)
- работать с данными через генераторы

##  Как работает программа

После запуска пользователь проходит следующие шаги:

1. Выбор источника данных (JSON / CSV / XLSX)
2. Фильтрация по статусу
3. Сортировка по дате
4. Фильтрация по валюте (опционально)
5. Поиск по описанию (опционально)
6. Вывод итогового списка транзакций

---

##  Основные модули

### `utils.py`

Загрузка данных из файлов:
- JSON
- CSV
- XLSX

Обработка ошибок:
- файл не найден
- неверный формат
- пустой файл

---

### `processing.py`

Работа с транзакциями:
- `filter_by_state` — фильтрация по статусу
- `sort_by_date` — сортировка по дате
- `process_bank_search` — поиск по описанию (через `re`)
- `process_bank_operations` — подсчёт операций по категориям

---

### `generators.py`

Генераторы:
- `filter_by_currency` — фильтрация по валюте
- ленивое выполнение (экономия памяти)

---

### `masks.py`

Маскирование:
- номера карт
- номера счетов

---

### `widget.py`

Форматирование:
- преобразование даты
- подготовка данных для вывода

---

### `external_api.py`

Работа с API:
- конвертация валют (USD / EUR → RUB)
- использование API-ключа из `.env`

---

### `decorators.py`

Декоратор `log`:
- логирование успешных вызовов
- логирование ошибок

---
