# PythonCourse8Project

## Описание проекта

**PythonCourse8Project** — учебный Python-проект, реализующий набор функций для обработки финансовых данных:
маскирование номеров карт и счетов, фильтрацию и сортировку операций.

Проект предназначен для практики:

* работы с функциями
* обработки списков и словарей
* модульной структуры Python-кода

---

## Описание модулей

### `masks.py`

Модуль содержит функции для **маскирования конфиденциальных данных**:

* маскирование номера банковской карты
* маскирование номера счёта

Используется для безопасного отображения финансовой информации.

**Пример результата:**

```
Visa Classic 6831 98** **** 7654
Счет **4305
```

---

### `processing.py`

Модуль отвечает за **обработку операций**:

* `filter_by_state` — фильтрация операций по статусу (например, `EXECUTED`)
* `sort_by_date` — сортировка операций по дате (по возрастанию или убыванию)

Работает со списками словарей, представляющих банковские операции.

---

### `widget.py`

Модуль объединяет функциональность проекта и предоставляет **удобный интерфейс**:

* форматирует данные операций
* использует функции из `masks.py` и `processing.py`
* подготавливает информацию для вывода пользователю

---

## ▶Пример использования

```python
from scr.processing import filter_by_state, sort_by_date
from scr.widget import get_date, mask_account_card

operations = [
    {"state": "EXECUTED", "date": "2019-08-26T10:50:58.294041"},
    {"state": "CANCELED", "date": "2018-12-23T11:47:52.403285"},
]

executed = filter_by_state(operations)
sorted_ops = sort_by_date(executed)

print(sorted_ops)
```
